1 上网速度慢

  • Ubuntu 网速慢，主要是把时间浪费在域名解析上。我们可以用dnsmasq解决这问题。

安装dnsmasq http://www.linuxidc.com/Linux/2010-06/26587.htm

1.   sudo apt-get install dnsmasq
2.  sudo gedit /etc/dnsmasq.conf     #resolv-file替换为 resolv-file=/etc/resolv.dnsmasq.conf 
3.  sudo cp /etc/resolv.conf /etc/resolv.dnsmasq.conf 备份
4.  sudo gedit /etc/resolv.conf     将其中的域名服务器全部去掉，加入以下这行   nameserver 127.0.0.1
5.   sudo gedit /etc/ppp/peers/provider（Ubuntu 10.04的文件名是provider，其他版本可能不同）  在 usepeerdns 前面增加 ＃ ，也就是把这条语句覆盖掉。 以防，resolv.conf的设置被pppoe覆盖。
6.  重启电脑。不重启，你会发现无法解析域名。所以一定要重启电脑，来达到重新启动dnsmasq的目的。

3. FTP搭建

  1. 安装ftp sudo apt-get install vsftp

  2. 建立Vsftpd服务的宿主用户： useradd vsftpd -s /sbin/nologin -s设置无效shell不支持登录(默认宿主root不方便)

  3. 建立ftp 账号： useradd ftptest -s /sbin/nologin -m 不能登录创建相应home

  4. 配置/etc/vsftpd.conf > 提前最好先备份。 详见http://blog.csdn.net/fafa211/article/details/8095081

  5. 目录设置

    sudo chmod a-w /home/ftptest   #撤销ftp账号根目录写权限
sudo mkdir /home/ftptest/data  
sudo chown -R ftpuser:nogroup /home/ftptest/data  # 保证该用户有写读权限

  6. 启动关闭 service vsftpd start/stop/restart
